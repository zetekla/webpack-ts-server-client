
Workflow:

`npm init -y` for quick init

`npm i -g webpack`

Usage:

`npm i`

`webpack`

`NODE_ENV=production webpack`

`npm run dev`   : run webpack in dev mode, with watch flag set it to watch the files

`npm run prod`  : run webpack in prod mode
